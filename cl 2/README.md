## 🔭 About

**SciCalc Pro** is a modern, professional-grade scientific calculator built entirely with vanilla HTML, CSS, and JavaScript. It features a stunning glassmorphism UI with dark/light theme support, full scientific functions, memory operations, calculation history, and complete keyboard support.

This project was created to demonstrate clean code architecture, modern CSS techniques, and interactive JavaScript development.

> **Developed by Tayyab**

## ✨ Features

### 🔢 Basic Operations
- ✅ Addition, Subtraction, Multiplication, Division
- ✅ Decimal support
- ✅ Percentage calculations
- ✅ Parentheses for complex expressions

### 🔬 Scientific Functions
- ✅ **Trigonometry**: sin, cos, tan (and inverses)
- ✅ **Logarithms**: log (base 10), ln (natural)
- ✅ **Powers**: x², x³, xʸ, eˣ
- ✅ **Roots**: √ (square root), ³√ (cube root)
- ✅ **Other**: factorial (x!), absolute value (|x|), 1/x
- ✅ **Constants**: π, e
- ✅ **Angle Modes**: DEG / RAD toggle
- ✅ **Modulo**: mod operation

### 💾 Memory Functions
- ✅ **MC** - Memory Clear
- ✅ **MR** - Memory Recall
- ✅ **M+** - Memory Add
- ✅ **M−** - Memory Subtract
- ✅ **MS** - Memory Store

### 🎨 UI/UX Features
- ✅ Premium glassmorphism design
- ✅ Dark / Light theme toggle
- ✅ Animated gradient background
- ✅ Smooth button animations & ripple effects
- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Auto-resizing display text
- ✅ Toast notifications

### 📜 History
- ✅ Full calculation history (up to 50 entries)
- ✅ Click to reload previous calculations
- ✅ Persistent history (saved in localStorage)
- ✅ Clear history option

### ⌨️ Keyboard Support
- ✅ Full keyboard input support
- ✅ Number keys, operators, Enter, Escape, Backspace
